# PulseFit Studio

PulseFit Studio is a simple and neat dark-themed fitness website built with HTML, CSS, and JavaScript. It uses a typewriter-style font, colorful line-art illustrations, and interactive sections to help users explore workout options and track basic fitness progress.

## Features Implemented

- Dark-themed responsive landing page
- Typewriter-style typography for a distinct visual identity
- Workout plans section with multiple fitness routines
- BMI calculator with instant result and category display
- Category-based exercise section for strength, flexibility, and cardio
- Progress tracking sliders for workouts, water intake, and sleep
- Local storage support to save progress in the browser
- Weekly class timetable section
- Custom line-art illustrations using SVG instead of image photos

## Project Structure

- `index.html` - Main webpage structure
- `styles.css` - Styling, layout, dark theme, and responsive design
- `script.js` - BMI calculator and progress tracking logic

## How to Run the Project

1. Download or clone the project files.
2. Make sure `index.html`, `styles.css`, and `script.js` are in the same folder.
3. Open `index.html` in any modern web browser.
